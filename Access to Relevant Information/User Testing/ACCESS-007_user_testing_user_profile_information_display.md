## **ACCESS-007:** User testing - User Profile Information Display  

> **Summary:** Verify that user profile information displays correctly and completely on profile page  <br>

**Preconditions:** User is logged in; User profile data exists  

Scenario 1: View own profile information

 | \# | Step | Expected Behavior | 
 |----|------|-------------------| 
 | 1 | Navigate to profile settings | User profile page loads with personal information |
 | 2 | Check profile image | Profile image or placeholder displays correctly |
 | 3 | Verify display name and email | Display name and email show at the top of the profile |
 | 4 | Check account information | Username, email, and password fields display correctly |
 | 5 | Verify profile information | First name, middle name, last name, suffix, and address display correctly |

**Post-conditions:**  
 - All profile information displays correctly
 - Sensitive information like passwords are masked
 - Information is grouped logically into Account and Profile sections
